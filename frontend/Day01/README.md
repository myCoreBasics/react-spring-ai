# React 전/후 비교 실습

## 목적
- Vanilla JS에서 DOM 직접 조작 방식 이해
- React에서 State 기반 선언형 UI 이해

---

## 폴더 구조
- `01-vanilla` : 브라우저에서 직접 실행 (DOM 직접 조작)
- `02-react` : React + State 기반 UI

---

## 실행 방법

### 1) Vanilla JS
브라우저에서 직접 파일 열기:

```bash
open 01-vanilla/counter-vanilla.html
# 또는 파일 탐색기에서 더블클릭
```

### 2) React

```bash
cd 02-react
pnpm install   # 또는 npm install, yarn install
pnpm run dev   # 또는 npm run dev, yarn dev
```
