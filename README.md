# react-lecture
React Spring AI 프로젝트 

### 전체 설치 명령어 

#### 프로젝트 초기 설정 (모든 패키지 설치)
pnpm install

#### 프로덕션 패키지만 설치
pnpm add react react-dom react-router-dom openai lucide-react

#### 개발 패키지 설치
pnpm add -D vite @vitejs/plugin-react typescript @types/node @types/react @types/react-dom eslint @eslint/js eslint-plugin-react-hooks eslint-plugin-react-refresh typescript-eslint globals

### 실행 방법:
##### pnpm dev      # 개발 서버 (http://localhost:5173)
##### pnpm build    # 프로덕션 빌드
##### pnpm lint     # 코드 검사
##### pnpm preview  # 빌드 미리보기
